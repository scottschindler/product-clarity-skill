# ProductClarity

ProductClarity is an agent skill that turns messy product ideas into clear, specific, buildable product specs.

It helps with:

- product ideas
- startup ideas
- app concepts
- feature specs
- MVP plans
- PRDs
- user flows
- launch plans
- Cursor-ready build prompts

## Install

```bash
npx skills add https://github.com/YOUR_USERNAME/productclarity-skill --skill productclarity
